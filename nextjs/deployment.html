    📝 6. deployment.md (Deploying Next.js Apps)

    🚀 Deploying Next.js Apps

    You can deploy Next.js apps to Vercel (recommended), Netlify, or other platforms.

    ✅ Deploy to Vercel

    vercel deploy

    ✅ Deploy to Netlify

    netlify deploy

    ✅ Deploy with Docker

    Create a Dockerfile
    # Dockerfile
FROM node:alpine
WORKDIR /app
COPY . .
RUN npm install
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
